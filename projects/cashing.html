Cashing Mobile Application

Technologies Used: Unity | C# | Game Development

Category: Educational Application | Game-Based Learning | Mathematics

Project Overview

The Cashing App was developed as an educational tool to support students in understanding basic mathematics concepts involving cash. The application includes interactive mini-games, such as drag-and-drop activities, that help users learn how to recognize currency values and perform simple calculations. By combining learning content with game elements, the application aims to improve user engagement and make the learning process more interactive and intuitive.

My Role

As a team member, I contributed to the educational content planning by identifying suitable learning materials for the application. I was also responsible for developing one of the drag-and-drop mini-games focused on learning money concepts, as well as supporting the project documentation.

Key Features

  • Interactive drag-and-drop learning games
  • Cash value recognition activities
  • Simple calcuation exercises
  • Beginner-friendly education interface
