Restoran Badhriah Attendance System Prototype

Technologies Used: AppSheet | Microsoft Excel

Category: Business System | Workflow Application | Prototype

Project Overview

The Attendance System Prototype was created as a practical solution for monitoring staff attendance in Restoran Badhriah. In addition to recording attendance information, the system also supports the calculation of draft wages based on employee attendance records. The aim of this prototype was to improve administrative efficiency and reduce manual monitoring work. The system was built using AppSheet, allowing rapid development of a functional workflow-based application.

My Role

As the team leader, I coordinated task distribution among team members and managed overall project progress. I conducted stakeholder interviews to gather system requirements, developed the full prototype of the attendance system, and handled documentation review, presentation, and system demonstration.
