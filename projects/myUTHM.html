MyUTHM Mobile Application Prototype

Technologies Used: Figma

Category: UI/UX Design | Mobile App Prototype

Project Overview

The MyUTHM Mobile App Prototype was developed as a redesign concept for the existing university mobile application. The project focused mainly on UI/UX improvements, including layout organization, usability, and overall visual appearance. Using Figma, the prototype was designed to provide a more modern and user-friendly experience for students interacting with university-related services through mobile devices.

My Role

As the team leader, I organized task allocation, monitored team progress, and collaborated with other members to develop the prototype in Figma based on the provided storyboard. I also conducted user testing and was responsible for the project presentation and documentation.

Key Features

  • Improved app layout
  • Cleaner navigation structure
  • Updated visual design
  • User-centered interface prototype
