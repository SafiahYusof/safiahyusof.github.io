SMK DOJ PPKI Mobile Application

Technologies Used: Mobile Application | Social Impact | Entrepreneurship

Category: Dart | Flutter | Google API

Project Overview

The PPKI Mobile App was developed for PPKI students at SMKDOJ, Johor, to help them promote and sell their products through a digital platform. The application aims to increase public awareness of these students’ work while also encouraging cyber entrepreneurship skills. By providing a mobile-based selling and promotion channel, the app supports both visibility and potential sales growth. This project combines technical development with meaningful social impact, and it was built using Flutter, Dart, and Google API integration.
