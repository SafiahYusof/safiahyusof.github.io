Our StationARy Shop Mobile Application

Technologies Used: Unity | Blender | Markerless AR | 3D Modeling

Category: Mobile Application | Augmented Reality | 3D Visualization

Project Overview

Our StationARy Shop was developed as an interactive mobile application for a stationery shop. The main feature of this project is the integration of markerless augmented reality, which allows users to view stationery items as 3D models in a real-world environment without relying on printed markers. This project combines multimedia and interactive technology to create a more engaging shopping experience. Blender was used to create and prepare the 3D assets, while Unity was used to integrate the AR functionality and overall app experience.

My Role

As a team member, I contributed to the development of the application by creating part of the 3D stationery models, designing button interfaces for the app, and assisting with the Unity-based AR setup. I also supported the documentation and presentation of the project.
