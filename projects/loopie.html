Loopie : Game-Based Learning Python

Technologies Used: Unity | Blender | Figma

Category: Educational Game | Game Development | Interactive Learning

Project Overview

Loopie is a game-based learning application developed to help beginners understand Python programming in a more interactive and enjoyable way. The application uses mini-games and hands-on coding practice to increase engagement and improve learning effectiveness. The project combines educational content with game design principles, creating an experience that is both fun and informative. Unity was used for development, Blender supported visual asset creation, and Figma was used to plan and design the user interface.

My Role

This was an individual project, and I was responsible for the entire development process independently. I handled the coding, design, testing, documentation, presentation, and all other aspects of the project from planning to completion.

Key Features

  • Python learning mini-games
  • Interactive practice activities
  • Beginner-friendly gameplay
  • Educational and engaging design

What I Learned

  • How game mechanics can support learning outcomes
  • How to combine programming education with interactive design
  • The importance of user engagement in educational applications
